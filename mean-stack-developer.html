MEAN/MERN Stack Developer

​We are looking for a MEAN Stack Developer to join our dynamic team. You will work with the Business Analyst and bridge the gap between requirements and technical implementation, taking an active role on both sides and defining how the application looks as well as how it works.

RESPONSIBILITTIES:
  • 4+ years Proven work experience as MEAN/MERN Stack Developer
  • Creating and implementing web applications using the MEAN/MERN stack technologies. This involves designing the overall architecture and developing the frontend and backend components.
  • Strong understanding of Kubernetes and containerization concepts.
  • Using Node.js and Express.js to handle server-side logic, application logic, and API development.
  • Working with MongoDB, a NoSQL database, to design and manage the database schema and handle data storage for the application.
  • Creating RESTful APIs to facilitate communication between the frontend and backend components of the application.
  • Working closely with other team members, such as UX/UI designers, backend developers, and project managers, to collaborate on the development process and ensure the successful delivery of the project.
  • Write comprehensive technical documentation and provide training and support to other team members as needed.
  • Using version control systems like Git to track changes and collaborate effectively with other team members.
  • Familiarity with cloud platforms (preferably on AWS) and containerization technologies.
  • Assisting in deploying the application to production environments and handling any issues that arise during or after deployment.
  • Staying updated with the latest developments in MEAN/MERN stack technologies and web development trends to incorporate the best practices and tools into the project.

PREFERRED:

  • Strong experience in building applications using AngularJS or ReactJS and React Native.
  • Strong experience in building applications using NodeJS and Kubernetes.
  • Strong understanding of databases and experience with MongoDB queries.
  • Proficiency in web markup, including HTML5, CSS, and JavaScript.
  • Proficient understanding of code versioning tools, such as Git, with the ability to manage multiple code branches.
  • Experience in Trade Directory domain is a good-to-have skill.
  • Experience in AWS is a good-to-have skill.
JOB LOCATION:

Onsite – Kosmo One, Chennai.

JOB TYPE:

Full Time
